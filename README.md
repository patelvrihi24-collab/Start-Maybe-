# START MAYBE?

A task manager that tracks not just deadlines, but how work actually progresses.

🌐 Live Demo: https://patelvrihi24-collab.github.io/Start-Maybe-/

## Features

* Smart task suggestions
* Deadline countdown timers
* Priority-based sorting
* Progress timeline (Started → Midway → Completed)
* Notes and file attachments
* Local storage persistence

## Why I Built This

Most task managers only track tasks. START MAYBE tracks the gap between planning and reality by recording how tasks evolve over time.
